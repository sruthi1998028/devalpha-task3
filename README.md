# Knowledge Quest 🚀
A responsive, interactive trivia application built with Vanilla JavaScript, HTML5, and CSS3. 

## Features
- **Dynamic Rendering:** Questions are generated dynamically from a JavaScript array of objects.
- **State Management:** Tracks user progress and score in real-time.
- **Modern UI:** Responsive design using CSS Flexbox for a clean, professional aesthetic.
- **Logic-Driven:** Implements complete CRUD-style state transitions (Question -> Scoring -> Completion).

## Tech Stack
- Frontend: HTML5, CSS3, Vanilla JavaScript (ES6+).
- Deployment: GitHub Pages.
